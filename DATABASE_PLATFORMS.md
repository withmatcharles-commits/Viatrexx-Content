### PL01: Facebook

| Content Type | Supported | Dimensions | Notes |
|-------------|-----------|-----------|-------|
| Static Post | ✅ | 1080×1080 | Square, community-focused caption |
| Carousel | ✅ | 1080×1080 | Up to 10 slides |

- Include follow-up question to encourage engagement
- Pin social proof / peer-verified testimonial in comments
- Golden Hour posting for engagement

**SEO Requirements:**
- H1: [Product/Topic] — [Primary Search Term]
- H2/H3 hierarchy following content architecture
- Meta description: [Problem] + [Solution] + [Viatrexx Difference]

- Schema markup (MedicalOrganization, Product, FAQ)
- Alt text on all images
- AEO: Direct answer in first 2 sentences for Google snippets
