## 2. Page Definitions

### Dashboard (`/dashboard`)

- **This Week Card** вҖ” Topic, pillar badge, phase-aware progress
- **Pipeline Summary** вҖ” Draft вҶ’ Review вҶ’ Scheduled вҶ’ Posted columns
- **Next 4 Weeks Strip** вҖ” Cards for weeks 5вҖ“8
- **Quick Stats Row** вҖ” Monthly totals, engagement trend, top post
- **Recent Activity Feed** вҖ” Last 10 actions with agent attribution chips
- **AI Agent Status Panel** вҖ” Claude, Jules, Gemini, NanaBanana indicators
- **API Health Strip** вҖ” Compact bottom bar: `Jules 12/15 в–Ҳв–Ҳв–‘в–‘ | NanaBanana 62/100 в–Ҳв–Ҳв–Ҳв–Ҳв–‘ | Claude вң… | Gemini вҖ” | Voice вҖ”`

Color coding: Green (<60%), Yellow (60вҖ“85%), Red (>85%), Gray (inactive).

### Topics Pages

**Pillars** (`/topics/pillars`) вҖ” 7 color-coded cards. Click expands to full detail with all topics. Actions: Add, Edit, Archive.

**Topic List** (`/topics/list`) вҖ” Filterable/sortable table. Columns: Title, Pillar, Audience, Status, Scheduled Week. Filters by pillar, status, audience, search.

**Schedule** (`/topics/schedule`) вҖ” Vertical week list with 5 zones: Posted (muted) вҶ’ In GHL (green) вҶ’ In Dashboard (blue) вҶ’ Next to Create (yellow) вҶ’ Future (default). Drag-drop reschedule.

**Topic Detail** (`/topics/[topic_id]`) вҖ” Topic data вҶ’ Briefs (Master/Type/Platform with agent attribution) вҶ’ Blog Posts (Phase 2) вҶ’ Content Pieces grid with product image badges.

### Drafts Pages

**Briefs** вҖ” Table with agent attribution (Claude/Jules). Status filters.

**Content** вҖ” Card grid. Each card: thumbnail, type badge, platform chips, image group labels ("FB+LI" / "IG" / "Pin"), product image indicator.

**Blogs** (Phase 2) вҖ” Table: Core vs. Companion badge, word count, SEO score, generated by Gemini.

**Content Detail** (`/drafts/[content_id]`) вҖ” Brief reference (collapsible) вҶ’ Asset preview by image group (Group A/B/C side by side) вҶ’ Product image indicator with GHL source label вҶ’ Platform previews (mockup tabs) вҶ’ Copy per platform вҶ’ AI Review results вҶ’ Action bar.

### Scheduled (`/scheduled`)

Calendar view (monthly) + List view. Color-coded by type and phase. GHL status indicators.

### Posted (`/posted`)

Card grid with metrics. Click for full lineage: Pillar вҶ’ Topic вҶ’ Brief вҶ’ Content вҶ’ Post вҶ’ Metrics.

### Analytics (`/analytics`)

Panels: Overview KPIs, By Platform, By Content Type, By Pillar, By Awareness Level, By Phase, Growth Trends. All filterable.

### Content Flow (`/content-flow`)

Timeline/feed view. Filters: Content Type, Platform, Status, Pillar, Phase. Sort by date.

### Settings (`/settings`)

**Brand Kit** вҖ” Colors, fonts, logo from brand kit JSON.

**Platform Connections** вҖ” GHL API status, social account mapping.

**GHL Products Panel:**
```
Products Cached: 99 | Last Sync: 2d ago
[Sync Now] [View Product List]
Auto-sync: Weekly (Sunday midnight)
```

**AI API Analytics Panel** вҖ” Per-provider monitoring cards:
- Jules: Daily tasks gauge, concurrent count, reset timer, 7-day sparkline, task breakdown
- NanaBanana: Daily images gauge, reset time, image breakdown by group, monthly cost
- Claude: RPM gauge, weekly calls/tokens, monthly cost vs budget
- Gemini: (Phase 2) Daily requests gauge
- ElevenLabs: (Phase 4) Monthly characters gauge
- Budget overview strip: Monthly AI spend vs budget

**Alert Config** вҖ” Toggle per provider: >80% daily, rate limit hit, >80% budget, auth error.

**Phase Control** вҖ” Enable/disable phases 1вҖ“4.

**Review Settings** вҖ” Max iterations, auto-approve, strictness.

**User Management** вҖ” Roles: Admin, Content Manager, Reviewer, Viewer.

## 3. Modals

| Modal | Trigger | Purpose |
|-------|---------|---------|
| Topic Quick-Add | "Add Topic" | New topic with pillar, concept, keywords |
| Schedule Topic | "Schedule" on topic | Assign to future week |
| Brief Generator | "Generate Brief" | Trigger Claude or Jules (shows agent) |
| Content Review | Review result | 16-point checklist with pass/warn/fail |
| GHL Push | "Send to GHL" | Confirm platforms, date/time, tags |
| Platform Preview | "Preview" | Device-frame mockups per platform |
| Image Group Preview | Click image | Side-by-side Group A/B/C comparison |
| API Limit Warning | Pre-flight check | Which provider, usage, recommended action |

## 4. Workflows

### Weekly Content Production (Phase 1)

```
Monday: Claude вҶ’ Master Brief вҶ’ Product image fetch (GHL) вҶ’ Jules 4 brief tasks
Tuesday: Jules 5 image tasks (NanaBanana + Remotion compositing)
Wednesday: Jules 4 tasks (carousels + captions)
Thursday: Jules 2 assembly tasks вҶ’ Claude reviews 7 pieces
  в”ңв”Җв”Җ PASS вҶ’ GHL Push (28 posts)
  в””в”Җв”Җ FAIL вҶ’ Jules revision (FriвҖ“Sun buffer)
```

### Review Checklist (16 checks R01вҖ“R16)

See AI_AGENTS.md for full checklist.
